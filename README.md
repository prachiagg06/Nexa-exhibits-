NEXA EXHIBITS



🌟 Overview
NEXA EXHIBITS is a modern, visually engaging website for an exhibition and event management company.
It showcases the company’s portfolio, services, and provides a contact interface for visitors to reach out.

✨ Features
Home Page – Attractive hero section with key company highlights.

Portfolio Page – Display of past exhibition setups and designs.

Services Section – Clear and concise listing of offerings.

Contact Page – Interactive form for visitor inquiries.

Responsive Design – Works smoothly across desktop, tablet, and mobile devices.

Modern UI/UX – Clean design with smooth transitions and hover effects.

🛠 Technologies Used
Frontend: React.js, HTML5, CSS3, JavaScript

Styling: Custom CSS

Animations & Effects: CSS transitions and hover effects

Package Management: npm / Node.js
