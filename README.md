# 🎨 NEXA EXHIBITS
**NEXA EXHIBITS** is a modern, responsive, and visually engaging website for an exhibition and event management company.  
It showcases the company’s **portfolio**, **services**, and provides an **interactive contact interface** for visitors. 

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Home Section** | Attractive hero images with company highlights. |
| 🖼 **Portfolio Page** | Showcase of previous exhibition setups and designs. |
| 💼 **Services Section** | Clear listing of company offerings. |
| 📬 **Contact Form** | Interactive frontend-only form for inquiries. |
| 📱 **Responsive Design** | Optimized for desktop, tablet, and mobile devices. |
| 🎨 **Modern UI/UX** | Clean design with smooth transitions and hover effects. |

---

## 🛠 Technologies Used

- **Frontend:** React.js, HTML5, CSS3, JavaScript  
- **Styling:** Custom CSS  
- **Animations & Effects:** CSS transitions, hover effects  
- **Package Management:** npm / Node.js  

---

## ⚡ Setup & Run Locally

1. **Clone the repository**
git clone https://github.com/prachiagg06/Nexa-exhibits-.git
cd nexa-exhibits

---
Install dependencies
npm install
---

Start the development server
npm start
---
Open in browser: http://localhost:3000

📌 Usage
Navigate through Home, Portfolio, Services, and Contact sections.
Fill the Contact Form to test frontend interactivity.
Customize images, content, and portfolio items as per your branding.

🚀 Future Enhancements
