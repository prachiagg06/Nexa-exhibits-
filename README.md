🎨 NEXA EXHIBITS
NEXA EXHIBITS is a modern, responsive website for an exhibition and event management company.
It showcases portfolios, services, and provides an interactive contact interface for visitors.

✨ Features
🏠 Home Section: Attractive hero images and company highlights
🖼 Portfolio Page: Showcase of previous exhibitions and designs
💼 Services Section: Clear listing of company offerings
📬 Contact Form: Interactive frontend-only form for inquiries
📱 Responsive Design: Optimized for desktop, tablet, and mobile
🎨 Modern UI/UX: Clean design with smooth transitions and hover effects


🛠 Technologies Used
Frontend: React.js, HTML5, CSS3, JavaScript
Styling: Custom CSS
Animations: CSS transitions, hover effects
Package Management: npm / Node.js

⚡ Setup & Run Locally
Clone the repository: git clone https://github.com/prachiagg06/Nexa-exhibits-.git
cd nexa-exhibits
Install dependencies

npm install
Start development server

npm start
Open in browser: http://localhost:3000

📌 Usage
Navigate through Home, Portfolio, Services, and Contact sections.
Test the Contact Form (frontend-only).
Customize images, text, and portfolio items to fit your brand.
